03 - A simple JavaScript program

Intro

Nesta aula veremos um pequeno problema simples para facilitar o entendimento do JavaScript e da lógica utilizada em programas básicos.

Variáveis

Podemos imaginar variáveis como pequenas caixas onde armazenamos informações. Neste exemplo, iremos definir três variáveis para guardar: nosso dinheiro, o valor da coxinha e o valor do pastel.

let

Usamos a palavra-chave let para criar uma variável. Após o let, definimos o nome da variável e o valor que será armazenado nela, como em let nome = 'David'. Assim, criamos uma variável chamada nome que guarda o valor David.

Exemplo usado em nosso código:

Variáveis utilizadas:

  • let meuDinheiro = 10;
  • let coxinha = 5;
  • let pastel = 7;
  • let troco = meuDinheiro - coxinha;

Entendendo melhor:

Utilizaremos essas variáveis para compreender o funcionamento de um programa simples. Neste caso, queremos verificar se é possível comprar o que desejamos com o valor disponível.

Para realizar essa verificação, utilizamos um recurso chamado condicionais, que permitem tomar decisões dentro do código.
